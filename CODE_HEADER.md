##Template
###Single author:

```cpp
// *Purpose of this file, a brief description of what is it supposed to do*
// Author(s):       *AuthorName* <*AuthorEmail*>
//
// Copyright (c) 2015-*CurrentYear* CtNorth Team
//
// Distributed under the MIT license (See accompanying file LICENSE.md or copy at
// https://opensource.org/licenses/MIT)


```

###Multiple authors:

```cpp
// *Purpose of this file, a brief description of what is it supposed to do*
// Author(s):       *AuthorName1* <*AuthorEmail1*>
//                  *AuthorName2* <*AuthorEmail2*>
//                  *AuthorName3* <*AuthorEmail2*>
//
// Copyright (c) 2015-*CurrentYear* CtNorth Team
//
// Distributed under the MIT license (See accompanying file LICENSE.md or copy at
// https://opensource.org/licenses/MIT)


```

##Examples:
###Single author:

```cpp
// Main Cat syncing code
// Author(s):       John <john@mywebsite.net>
//
// Copyright (c) 2015-2017 CtNorth Team
//
// Distributed under the MIT license (See accompanying file LICENSE.md or copy at
// https://opensource.org/licenses/MIT)
```

###Multiple authors:

```cpp
// Custom implementation of MD5

//

// https://opensource.org/licenses/MIT)
```
